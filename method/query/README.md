## 查询对应信息方法
### get_starSign
获取对应星座<br>
传入列表 格式为 月,日<br>
返回字符串

```python
get_starSing([12, 24]) # 魔蝎座
```

### get_zodiac
获取对应生肖<br>
传入某年<br>
返回字符串

```python
get_zodiac(2020) # 鼠
```

### get_day
获取和计算天数天数<br>
传入列表 年,月,日<br>
返回整数类型

```python
get_day([2020, 11, 0], [2020, 11, 8]) # 8
```
PS:对应的月非自然月数，而是 365 / 12 得出的所以计算会略有不准，可以做估计的数。不准主要在月单位上。
